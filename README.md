# WaltonsCreativeApp
Walton’s Creative Studio – Financial Management System

Tech Stack:
- Node.js + Express
- SQL Server (WaltonsCreativeDB)
- HTML, CSS, JavaScript

How to Run:
1. Install Node.js
2. Run: npm install
3. Configure .env with DB credentials
4. Start server: npm start
5. Open browser at http://localhost:3000

Roles:
- Owner
- Manager
- Accountant

This system supports:
- Role-based dashboards
- Expense tracking with audit logging
- Invoice and payment management
- Secure session-based authentication
